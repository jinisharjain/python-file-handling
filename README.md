Python File Handling

This repository contains Python programs that demonstrate how to work with files. File handling allows programs to store data permanently and retrieve it later.

These programs practice basic operations such as writing to a file, reading from a file, and appending data to an existing file.

Programs Included
1. Write to File

A program that takes user input and saves it into a file.

Concepts used:

File creation

Writing data to files

2. Read from File

A program that reads the content of a file and displays it to the user.

Concepts used:

File reading

Displaying stored data

3. Append to File

A program that adds new information to an existing file without deleting the previous data.

Concepts used:

Appending data to files

File handling modes

4. Contact Book

A simple program that stores contact names and phone numbers in a file so they can be saved and accessed later.

Concepts used:

File storage

User input

Data persistence

Purpose of This Repository

The goal of these programs is to practice file handling in Python and understand how data can be stored and retrieved from files in real applications.
